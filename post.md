# LinkedIn Post Draft - CodePilot

Built something I wanted for my own workflow: **CodePilot**.

CodePilot is a local-first CI/CD dashboard inspired by GitHub Actions, powered by `nektos/act`, and focused on developer speed.

Instead of jumping across multiple tools, it gives one place to:

- Authenticate with GitHub (repo-layer permissions)
- Sync repositories and discover workflow files
- Run GitHub Actions locally with `act`
- Watch live queue and run status updates
- Stream logs in real time
- Manage encrypted env profiles for workflows

The goal is simple: **faster feedback loops for CI/CD while staying local-first**.

It is still in development, and I am actively improving reliability, UX, and workflow observability.
